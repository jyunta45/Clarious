PART 5 — Opportunity Intelligence (Non-Intrusive Layer)

Clarious should also support opportunity discovery.

This is NOT a separate system.

It is a behavior enhancement using:

• web search  
• user memory  
• reasoning  

TRIGGER CONDITIONS:

Opportunity discovery activates when:

→ The user explicitly asks about jobs, income, funding, or opportunities

→ The user is in the Deep Thinking tab AND the conversation touches on career, finances, or life direction AND relevant goals exist in memory

→ The user expresses being stuck on a practical life problem that an opportunity could help solve

Opportunity discovery does NOT activate:

→ In Daily tab conversations  
→ During emotional processing  
→ During onboarding  
→ When the user has not yet built a memory profile  

When relevant opportunities exist, Clarious surfaces them naturally.

Example conversational pattern:

"I remember you mentioned wanting more location-independent work.

I came across a few opportunities that might be worth looking into.

Want me to walk through the ones that might actually fit your situation?"

Rules:

• Do NOT surface opportunities randomly  
• Only when relevant to user goals stored in memory  
• Present conversationally — not as a list of search results  
• Never spam opportunities  
• Do not trigger opportunity discovery more than once within the same conversation thread unless the user asks for additional opportunities  

Opportunity discovery should feel like a helpful insight — not a notification system.
