STEP 3 — Implement Webhook Logic

Inside the route:

1. Get signature from headers:
const sig = req.headers["stripe-signature"];

2. Verify event:

  process.env.STRIPE_WEBHOOK_SECRET
);

Handle error properly:
→ return 400 if signature fails

Implement the following cases:

CASE 1 — checkout.session.completed

  const session = event.data.object;

- Extract userId from:
  session.metadata.userId

- If userId exists:
  Update user in database:

  tier = "partner"
  tierUpdatedAt = current ISO string
  stripeCustomerId = session.customer
  stripeSubscriptionId = session.subscription

Use existing db + users schema
Do not change schema structure

---

CASE 2 — customer.subscription.deleted

- Get subscription:
  const subscription = event.data.object;

- Find user where:
  stripeSubscriptionId === subscription.id

- Update:
  tier = "free"
  tierUpdatedAt = current ISO string
